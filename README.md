# &#9836; WwMusic
A very simple (work-in-progress) audio instrument and midi player module. Depends on [MidiPlayerJS](https://www.npmjs.com/package/midi-player-js).

## build, run, test
```sh
npm install
npm run build
npm run test

cd tools

node ./test-midi-player-js.js

node ./test-MidiToMediaPlayer.js
```
